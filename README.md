## ✨ الميزات الرئيسية

### 🏗️ إدارة المشاريع
- **عرض كامل للمشاريع** مع حالات متعددة (نشط / متوقف / مكتمل / معلق)
- **تتبع الميزانية** بوقت فعلي مع تنبيه عند تجاوز الحد
- **نسبة الإنجاز** مع شريط تقدم بصري
- **تفاصيل كاملة** تشمل الموقع، المدير، العمال، تواريخ البداية والنهاية
- **إضافة/تعديل المشاريع** عبر نموذج ذكي يدعم العمل أوف لاين

### 👷 إدارة العمال
- **قائمة العمال** مع تصفية حسب الحالة والمشروع
- **بطاقة عامل تفصيلية** تشمل: الجنسية، الهوية، الراتب، المهارات
- **نسبة الحضور** محسوبة تلقائياً مع شريط تقدم
- **روابط بالمشروع** لعرض عمال مشروع بعينه

### 💰 المصروفات
- **تتبع المصروفات** مع تصنيفات متعددة (مواد / رواتب / صيانة / معدات)
- **سير العمل للموافقة**: معلق → موافق عليه / مرفوض
- **إجمالي سريع** للمصروفات المعتمدة في رأس الصفحة
- **تفاصيل كاملة** تشمل طريقة الدفع ووجود الإيصال

### 📦 العهد والمواد
- **تسجيل العهد** المُسندة للعمال وتتبع حالتها
- **قائمة العناصر** مع الكميات والقيم وحالة كل عنصر
- **حالات متعددة**: نشط / مُعاد / جزئي

### 🏦 الحسابات والخزينة
- **عرض أرصدة الحسابات** البنكية المتعددة
- **إجمالي الرصيد** مع عرض بصري جذاب
- **سجل المعاملات** مع تمييز الإيداع/السحب

### 📓 السجلات اليومية
- **تسجيل يومي** للأعمال المنجزة في الموقع
- **إحصائيات الحضور**: الحاضرون / الغائبون / ساعات العمل
- **الطقس والمواد** المستخدمة في كل يوم
- **المشاكل والتنبيهات** مع إمكانية إرفاق الصور
- **المهام المنجزة** بقائمة منظمة

### 📊 التقارير والإحصائيات
- **تقرير المشاريع**: توزيع الحالات، متوسط الإنجاز، نظرة ميزانية
- **التقرير المالي**: تفصيل المصروفات بالفئة، ملخص الموافقات
- **تقرير العمال**: الحضور التفصيلي، إجمالي الرواتب
- **3 تبويبات** للوصول السريع لكل نوع من التقارير

### 🔔 الإشعارات
- **إشعارات مصنفة**: نجاح / تحذير / خطأ / معلومات
- **عداد غير المقروء** في أيقونة الناف بار
- **تحديد الكل كمقروء** بضغطة واحدة
- **إشعارات Push** عبر Firebase FCM

## 🛠️ Tech Stack

| الطبقة | التقنية | الغرض |
|--------|---------|-------|
| **Frontend** | Flutter 3.x | واجهة iOS و Android |
| **State Management** | BLoC / Cubit | إدارة الحالة |
| **Navigation** | go_router | التنقل بين الشاشات |
| **Local Storage** | Hive + SharedPreferences | التخزين المحلي الأوف لاين |
| **Cache** | Hive (TTL-based) | نسخ من استراتيجية Redis |
| **HTTP Client** | Dio | التواصل مع الـ API |
| **Backend** | ASP.NET Core 8 | REST API |
| **Database** | SQL Server + EF Core | قاعدة البيانات الأساسية |
| **Cache Server** | Redis | تخزين مؤقت على الخادم |
| **Push Notifications** | Firebase FCM | الإشعارات الفورية |
| **DI** | GetIt | Dependency Injection |
| **Architecture** | Clean Architecture | تنظيم الكود |
| **Security** | JWT + 2FA | المصادقة والأمان |
| **Infrastructure** | Docker + Kubernetes | النشر والتوسع |
| **DDoS Protection** | Cloudflare | الحماية من الهجمات |
| **Search** | Elasticsearch | البحث المتقدم |

## 🏛️ المعمارية (Clean Architecture)

```
lib/
├── core/                          # البنية التحتية المشتركة
│   ├── constants/                 # الثوابت والنصوص
│   ├── di/                        # Dependency Injection (GetIt)
│   ├── errors/                    # Exceptions و Failures
│   ├── network/                   # Dio Client + Interceptors
│   ├── services/                  # NotificationService، SyncService
│   └── theme/                     # AppTheme (Light + Dark)
│
├── features/                      # كل ميزة منفصلة
│   └── [feature_name]/
│       ├── data/
│       │   ├── datasources/       # Remote API + Local Hive
│       │   ├── models/            # JSON ↔ Entity mappers
│       │   └── repositories/     # Repository Implementation
│       ├── domain/
│       │   ├── entities/          # كيانات النطاق النظيف
│       │   ├── repositories/      # واجهات Repository
│       │   └── usecases/          # حالات الاستخدام
│       └── presentation/
│           ├── bloc/              # BLoC + Events + States
│           ├── pages/             # شاشات Flutter
│           └── widgets/           # مكونات خاصة بالميزة
│
├── shared/
│   ├── models/                    # كيانات مشتركة (entities.dart)
│   ├── services/                  # SampleData
│   └── widgets/                   # مكونات مشتركة
│
└── router/
    └── app_router.dart            # go_router configuration
```
